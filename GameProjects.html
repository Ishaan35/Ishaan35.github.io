Game Programming Projects

3D Mario Kart in Unity (C#)



This is a 3D Mario game made in Unity using C#. I am a big fan of mario games with Mario Kart being one of my favourites as it is the most fun. I wanted to attempt making a small personal Mario Kart Game on my own and it is turning out to be a fun and interesting challenge. The game also has AI racers which the player can race against. Not all of the modelling is my own work, and certainly most of the assets belong to Nintendo. However, all of the programming and animations present in the game are my own work.

This is a 3D platformer game inspired by Mario created with Unity and C#. In this game, there are a few levels which I created from 3D models which were made by Nintendo. The player has to move around and destroy enemies, get coins, and find some interesting challenges along the way. The modelling is not entirely my work, however all the animations and programming is my own work.

Super Highway Runner (C#)



This is one of my first 3D games that I created with Unity and C#. In the game, the player has to select a car and then survive as long as possible on the road without touching any of the cars in the traffic. It is an endless runner which gets more difficult the longer you survive by makng your car faster. The game ends as soon as you bump into another car. The controls are touch screen and the player moves through the lanes. There is another game mode which uses tilt controls which utilizes the gyroscope on your phone and is more difficult than the standard mode. The game saves your high score, your cash, and your purchased cars with the in-game currency as well. This game currently has hundreds of reviews and downloads.

Raining Cubes (C#)



This is another 3D mobile game that I have created with Unity and C#. This game features a rolling sphere which has to avoid falling objects by rolling across the stage using the phone's gyroscope. There are a few themes which the player can choose from depending on his/her style and the player's highscore is recorded for future reference. This game has hundreds of downloads and reviews on the Play Store as well.

Fireworks Simulation (Java)



This is a project made in Processing (A language based on Java). The purpose of this project is to visualize and simulate different firework effects. This project features a starry sky with continuous fireworks being generated with all sorts of different colors, visual effects, and audio effects. There is also a nice meteor shower playing in the background.
